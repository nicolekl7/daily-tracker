# Nicole OS

Personal wellness, cycle intelligence, and wedding tracker.

## Stack
Single-file React app (Babel standalone). No build step. No dependencies to install.  
All data persists in browser `localStorage` — private, offline-capable.

## Deploy
1. Push this repo to GitHub
2. Import into Vercel → it detects static automatically
3. Done. No build command, no output directory needed.

## Local
Just open `index.html` in Chrome. That's it.
